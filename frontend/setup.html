🚀 College Events System - Setup Complete!

Your full-stack college events management system is ready. Follow these steps to start:

📋 Prerequisites

  • Node.js installed
  • MongoDB running (local or Atlas)
  • Internet connection for dependencies

🔧 Installation & Setup

  1. Open terminal in mini_project folder
  2. Install dependencies: npm install
  3. Install backend deps: cd backend && npm install
  4. Return to root: cd ..

🗄️ Database Setup

  1. Make sure MongoDB is running
  2. Update backend/.env if needed
  3. Default connection: mongodb://localhost:27017/college_events

▶️ Start the System

Option 1 - Both servers together:

npm run dev

Option 2 - Separate terminals:

Backend: cd backend && npm run dev

Frontend: cd frontend && npx live-server --port=3001

🌐 Access Your Application

  • Frontend: http://localhost:3001
  • Backend API: http://localhost:3000
  • Health Check: http://localhost:3000/health

🎯 Key Features

  • ✅ Event creation and management
  • ✅ Categories: Coding, Non-coding, Non-ANITS, Other
  • ✅ User authentication with Google
  • ✅ Admin panel for event management
  • ✅ MongoDB database integration
  • ✅ Responsive modern UI
